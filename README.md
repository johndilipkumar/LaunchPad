# 🚀 LaunchPad - Premium Launch Countdown Timer

A modern and visually stunning launch countdown website built with React, Vite, Tailwind CSS, and Framer Motion.

LaunchPad is an upgraded version of the classic Frontend Mentor Launch Countdown Timer challenge, featuring glassmorphism UI, smooth animations, interactive countdown cards, developer testing controls, confetti celebrations, and a fully responsive design.

## ✨ Features

- ⏳ Real-time countdown timer
- 🔄 Animated flip-card countdown transitions
- 🎉 Confetti celebration on launch
- 🌌 Animated cosmic background effects
- 💎 Glassmorphism user interface
- 📧 Email subscription form
- 📱 Fully responsive design
- 🎨 Modern gradient and neon-inspired styling
- 🚀 Framer Motion animations
- 🔧 Developer testing panel
- ♿ Accessibility-friendly structure

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/LaunchPad.git
```

Navigate into the project:

```bash
cd LaunchPad
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

---

## 🎯 Countdown Functionality

The countdown automatically updates every second and displays:

- Days
- Hours
- Minutes
- Seconds

When the countdown reaches zero:

- Countdown disappears
- Launch message appears
- Confetti animation is triggered
- Celebration screen is displayed

---

## 🧪 Developer Controls

LaunchPad includes a hidden testing panel for quickly testing launch states.

Developer panel options:

- Launch in 10 seconds
- Trigger instant launch
- Reset countdown

Perfect for testing animations and launch transitions during development.
